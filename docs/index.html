E-Commerce website

An E-Commerce website built with a combination of Vue and Tailwind on the Front-End and Python (Flask) and MongoDB Atlas on the server-side. I decided to build this project to test my Full-Stack development skills and learn a couple new technologies. The website includes a full-fledged store with items stored in the database and fetched through the API on the Flask server, Authentication and Authorisation using JWT as well as Stripe to handle the payments.

Vue (Vuex) | Tailwind | Flask | MongoDB Atlas | Stripe
